# Structure de la Page d'Accueil — NEXGEN SPORT

## 1. Fiche Talent (Fiche Produit)
La "fiche produit" sur la page d'accueil représente chaque talent sportif de manière claire, visuelle et orientée scouting.

### **Exemple de structure d'une fiche talent**
```
┌──────────────────────────────────────────────┐
│ [ PHOTO DU TALENT ]                          │
│                                              │
│ NOM & PRÉNOM                                  │
│ Sport • Poste / Catégorie                      │
│ Pays / Ville                                   │
│ Âge | Niveau                   │
│                                              │
│ ⭐ Stats / Résultats clés (3 à 5 max)          │
│ • Matchs / Tournois                           │
│ • Performances                                 │
│ • Classement / Récompenses                     │
│                                              │    │
│                                              │
│ [ VOIR LE PROFIL ]                            │
└──────────────────────────────────────────────┘
```

### **Points clés**
- Design moderne type *scouting / stats sportives*
- Toujours 1 action principale : **Voir le Profil**
- Présente l'essentiel en 5 secondes
- Optimisée pour jeunes sportifs (13-30 ans) + recruteurs

---

## 2. Barre de Recherche — Version Officielle NEXGEN SPORT
Permet aux visiteurs, clubs et agents de trouver rapidement un talent.

### **Structure (Desktop)**
```
────────────────────────────────────────────────────────────────────────────
🔍 TROUVER UN TALENT

[ ⚽ Sport ▼ ]   [ 🎯 Poste / Catégorie ▼ ]   [ 🌍 Pays / Ville ▼ ]  [   Rechercher un talent   ]

                           
────────────────────────────────────────────────────────────────────────────
```

### **Filtres adaptés au sport**
- **Sport** : Football, Basketball, Handball, Tennis, Sports de combat
- **Poste / Catégorie** : dépend du sport sélectionné
- **Pays / Ville** : localisation du sportif
- **Niveau** : Amateur, Semi-pro, Professionnel, Centre de formation
```

### **Style visuel (identité NEXGEN SPORT)**
- Icônes type Lucide
- Bords arrondis 
- UI type *data sport / scouting*

---

## 3. Intégration sur la Homepage

La Page d'accueil suit cet ordre :

1. **Hero minimal + slogan + Barre de recherche “Trouver un talent**
3. **Listing des talents (fiches produit)**
4. **Articles / vidéos du média NextGen (option)**
